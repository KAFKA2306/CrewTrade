# 貴金属スプレッド (Precious Metals Spread) 分析レポート

## エグゼクティブサマリー (Executive Summary)
純金上場信託などの貴金属ETFを用いたスプレッド取引（裁定取引）の分析結果です。
特筆すべき点として、**Gap（乖離）が一定水準（8%以上）まで拡大した後の平均回帰性は極めて高く（成功率 98%以上）**、強力なトレードシグナルとなることが確認されました。特に「1540.T (純金上場信託)」は、乖離からの回復日数が短く（中央値1〜3日）、資金効率の良い銘柄として特定されています。

## 分析カバレッジ
- **期間**: 2023-11-24 〜 2025-10-17
- **シグナル総数**: 128回
- **対象銘柄**: 1540.T, 1541.T, 1542.T, 1543.T

## 主要な発見
### 1. 高い収益性を示すGap基準
乖離が8%以上に拡大してから4%未満に戻るトレードの成功率はほぼ100%でした。

| 閾値 (Threshold) | 銘柄 | 成功率 (Success %) | 平均回復日数 |
|---|---|---|---|
| **Gap ≥ 8% → < 4%** | 1540.T | **100.00%** | 66.29 |
| | 1541.T | 98.92% | 93.16 |
| | 1542.T | 98.99% | 75.49 |

### 2. 回復スピードの速さ (|z| < 2.0 への回帰)
統計的な乖離（z-score）が2.0を超えた後、正常範囲に戻るまでの日数は非常に短いです。
- **1540.T**: 中央値 1.00日
- **1542.T**: 中央値 1.00日

これは、貴金属市場における一時的な価格歪みが、市場参加者によって迅速に修正されていることを示唆しています。
