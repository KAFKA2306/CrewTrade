# Index ETF 比較分析レポート

## エグゼクティブサマリー (Executive Summary)
本レポートでは、12種類の主要なインデックスETF（国債、高配当株、新興国株、コモディティなど）を対象に、詳細な比較分析を行っています。
各ETFのリスク・リターン特性、配当利回り、および長期的な保有適性を評価し、ポートフォリオ構築のための基礎データを提供します。特に**MSCIジャパン高配当（9銘柄）**や**新興国債券（6銘柄）**など、インカムゲインを重視したアセットの分析が充実しています。
