# 貴金属スプレッド・効率性および相関分析レポート

## 1. 貴金属スプレッドの資金効率 (ROI Efficiency)
単なる「勝率」ではなく、「期間収益率（ROI/Day）」を算出し、資金拘束を考慮した投資効率を評価しました。

### 主要な発見
**「1540.T (純金上場信託)」** が圧倒的な資金効率を示しています。特に**乖離が10%以上に拡大するのを待つ**ことで、投資効率（年率換算ROI）が劇的に向上することが判明しました。

| Scenario (乖離率) | 平均利益 (Est.) | Ticker | 平均日数 (Days) | 1日あたりROI | 年率換算 (Simple) |
|---|---|---|---|---|---|
| **12% → 6%** | 6.0% | **1540.T** | **5.00** | **1.20%** | **300.0%** |
| **10% → 5%** | 5.0% | **1540.T** | **5.00** | **1.00%** | **250.0%** |
| 12% → 6% | 6.0% | 1541.T | 22.38 | 0.27% | 67.0% |
| 12% → 6% | 1543.T | 6.0% | 37.70 | 0.16% | 39.8% |
| ... | ... | ... | ... | ... | ... |
| **8% → 4%** | 4.0% | 1540.T | 66.29 | 0.06% | 15.1% |

### アクション・インサイト
- **Wait for Deep Value**: 乖離率8%でのエントリーは、資金が平均2ヶ月以上（66日）拘束され、年率15%程度のリターンにとどまります。
- **Target 10%+**: 乖離率10%または12%まで待つことで、**わずか5日間で5-6%のリターン**（年率250%超）を獲得できるチャンスがあります。トレード頻度は減りますが、資金効率は20倍近く跳ね上がります。

## 2. 証券担保ローン戦略との相関 (Correlation Analysis)
メインの「証券担保ローン戦略 (SCL)」と「貴金属スプレッド (PM)」の同時運用リスクについての定性評価です。

### 2024-2025年の重複期間における観察
データが重複する2024年〜2025年の期間において、両戦略は共にプラスのパフォーマンスを示しました。

- **SCL (2024)**: Return +11.28%, Volatility 9.1% (Bullish)
- **PM (2024)**: 1540.T/1541.T 共に複数の収益機会あり（勝率ほぼ100%）

### 結論と注意点
- **現状は「順相関」の可能性**: 上昇相場（2024年）においては、両戦略とも利益を積み上げており、分散によるリターン底上げ効果が確認されました。
- **ストレス時の検証不足**: SCLが苦戦した2009年や2022年の期間において、PMのデータが存在しないため、**「暴落時のヘッジ機能」は未証明**です。
- **推奨**: PM戦略は「ヘッジ（保険）」としてではなく、**「独立したアルファ（収益機会）」**として扱い、SCLのリスク許容度とは別枠で資金管理を行うことを推奨します。
