## エグゼクティブサマリー (Executive Summary)

ここでは、メインの証券担保ローン戦略とは異なる、**多様なアルファ（超過収益）の源泉**を探求しています。

検証対象は、著名投資家の模倣（井村ファンド）から、統計的裁定（貴金属・利回りスプレッド）、そして実験的なAI予測（オラクル）や外部情報の活用（ブログリサーチ）まで多岐にわたります。
これらのユースケースは、単体での戦略化だけでなく、ポートフォリオ全体のリスク分散や、市場局面に応じた**「サブモジュール」としての活用**を目的として蓄積されています。
