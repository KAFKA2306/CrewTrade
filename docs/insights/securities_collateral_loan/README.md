# 証券担保ローンに関する知見

このディレクトリには、証券担保ローン戦略の分析およびバックテストレポートが含まれています。

## アクション・インサイト (Actionable Insights)

データ分析に基づく具体的な次なるアクションと提言です。


### 2. レジームチェンジへの対応策導入
2019-2020年のような市場環境の変化（レジームチェンジ）において、過去データの予測力が著しく低下しています。
- **Action**: 単純なヒストリカルデータだけでなく、マクロ経済指標やVIX指数などを用いた **「レジーム判定ロジック」** を組み込み、危険域では自動的にレバレッジを縮小する仕組みが必要です。

## 主な発見事項 (Key Findings)
- **Forward Volatility Risk**: バックテスト期間の制約（12-15%）に対し、フォワード期間のボラティリティは一貫して超過する傾向があります（実現ボラティリティは頻繁に19%を超えています）。

## 過去のバックテストパフォーマンス

各年度のバックテスト結果の集計です。
- **Backtest Return/Vol**: 最適化期間（過去5年間）の指標。
- **Forward Return/Vol**: その翌1年間の実現指標。
- **Status**: フォワード期間のボラティリティが制約を超過したかどうか。
